# 📦 Product Return Prediction System

A Machine Learning web application built with **Python**, **Streamlit**, and **Decision Tree Classifier** to predict whether a customer is likely to return a product.

## 🚀 Features

- 📊 Predict product return status
- 🤖 Decision Tree Classifier
- 📈 Model Accuracy Display
- 📋 Dataset Viewer
- 📊 Dataset Statistics
- 🔍 Missing Value Checker
- 📝 User-friendly Input Form
- ⬇️ Download Dataset

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-learn

## 📂 Dataset Features

- Product Price
- Delivery Days
- Customer Rating
- Purchase Count

### 🎯 Target

- Return Status
  - 0 = Product Not Returned
  - 1 = Product Returned

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/product-return-prediction-system.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
